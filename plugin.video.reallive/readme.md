# Reallive for kodi 0.1.1
## 简介
- Reallive是一个直播聚合插件，Real取自对本插件帮助巨大的开源项目wbt5大佬的Real-url，以表示对该开源项目的感谢！更多直播网站和功能后续支持
## 主要功能
适配网站 | 输入房间号/链接进入直播 | 按分类浏览直播列表  |  分类浏览支持翻页  |  搜索直播间和主播  |  搜索列表支持翻页 | 作者 | 更新时间
