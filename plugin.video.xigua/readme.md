# 西瓜视频 for kodi 0.1.0
## 简介
- 西瓜视频是字节跳动旗下视频平台，通过个性化推荐为您提供新鲜、好看的视频内容，并帮助视频创作人们轻松地向全世界分享自己的视频作品，更有原创综艺节目等你来看
## 主要功能
> - [x] 视频分类
> - [x] 在线播放视频，支持选择分辨率，最高720p
> - [x] 使用官方api解析视频
> - [ ] 搜索功能
> - [x] 缓存功能（减少向服务器请求次数）
> - [x] 从网络粘贴板导入西瓜视频链接进行解析
> - [x] 输入西瓜视频i号（纯数字）或者album号（字母数字混合）解析视频
