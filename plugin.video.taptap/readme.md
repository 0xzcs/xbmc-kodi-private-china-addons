# taptap for kodi 0.1.0
## 简介
- TapTap是一个推荐高品质手游的手游分享社区，实时同步全球各大应用市场游戏排行榜，与全球玩家共同交流并发掘高品质手游。每一款推荐游戏，都是由专业的测评团队从全球海量的游戏中精选而出，只为你提供好玩的手机游戏
## 主要功能
> - [x] 编辑推荐视频，为你推荐视频
> - [x] 游戏排行榜
> - [x] 显示官方发布的视频和玩家上传的视频
> - [x] 在线播放视频，支持任意选择分辨率，最高可播放4k60fps
> - [x] 使用官方api解析视频
> - [ ] 搜索功能
> - [ ] 缓存功能（减少向服务器请求次数）
