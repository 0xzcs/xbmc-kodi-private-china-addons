# bilibili for kodi 0.4.0
## 简介
- bilibili是国内知名的视频弹幕网站，这里有最及时的动漫新番，最棒的ACG氛围，最有创意的Up主。大家可以在这里找到许多欢乐
## 主要功能
> - [x] 视频分类
> - [x] 在线播放1080p视频
> - [x] 使用官方api和第三方api解析视频
> - [x] 在线播放番剧（540p，6分钟试看）
> - [x] 搜索功能(使用官方搜索api，支持输入bv号直达视频)
> - [x] 缓存功能（减少向服务器请求次数）
> - [ ] 实现显示弹幕功能
